Third-Party Notices

Effective date: 26 January 2026

This document lists the main third-party services used to operate Plaspa. This is not a replacement for those providers’ own terms and policies.

1. Hosting & Infrastructure

  • Google Cloud Run: Hosts our primary backend service.
  • Google Cloud Logging: Stores operational logs for security and monitoring.
  • Google Artifact Registry: Manages container images for deployment.

2. Crash Reporting

  • Sentry: Used to collect crash and error reports to improve stability. Sentry receives diagnostic events when the App encounters an error.

3. Open-source Software

The App includes open-source libraries. Licensing and attribution notices are available within the App Settings under "Third-Party Licences".
